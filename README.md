These are the MATLAB files I made for my final project in MAE 247 at UCSD. The project was a report on the paper Geometry-Aware Edge-State Tracking for Robust Affine Formation Control.
To get the figure of the RKF and GARKF comparisons at differing lambda values in the paper, run sim_est_rkf_garkf.m then plot_est_rkf_garkf.m
To get the hexagon formation simulation, run the plot_target_traj.m file
To get the new simulation that runs a 4x4 grid formation, run the sim_grid_blackout.m then the plot_grid_blackout.m files
